# Sharp Surv

Just a fun little project i'm currently doing in C# & Raylib. 

I'm mostly doing this as a C# & Raylib learning type of thing so yeah. 
I'm planning for this to one day be some sort of very simple open-source 2D survival/sandbox game.


## Installation

Download [.NET SDK 8.0](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

Clone the repo.

```bash
  git clone https://github.com/kikinndev/SharpSurv.git
  cd SharpSurv
```

Run these to build the project & install raylib-cs, rlImGui-cs.
```bash
  dotnet build
  dotnet package add raylib-cs
  dotnet package add rlImGui-cs
```

Open any C# code editor and open the solution. Run with F5.
